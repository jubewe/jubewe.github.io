Token Help

Commands

    Regular

    1. bot/ping > Shows Info about the Bot, including ping, channels, logchannels and handels
    2. channels > Shows in how many channels the bot is in

    Depends on Permissions

    1. stats ~user/all\self~ ~channel\currentchan~ ~time\logstart~ ~-filters\modfilters~ > description
    2. stats ~public/global~ ~user\self~ ~-filters~ > Shows all logstats of a user filtered by custom filters and time
    3. log ~chan~ > starts logging a channel
    4. unlog ~chan~ > description

    Admin/Owner

    1. num > gets the current number in the console - useful for finding errors
    2. off > stops the script without restart (manual restart required)
    3. join ~chan~ > joins a channel and handles messages
    4. leave ~channel~ > leaves a channel
    5. say ~message~ > says ~message~
    6. send ~channel~ ~message~ > sends ~message~ in ~channel~
    7. test > test command
    8. channels ~(-force)~ > gets the current channels (-force sends the names in chat)
    9. getlogs ~(-force)~ > gets the current logging channels (-force sends the names in chat)
    10. addperm ~user~ > adds perms to a user so it can use the commands
    11. removeperm ~user~ > removes perms from a user
    12. checkperm ~user\self~ ~channel\currentchannel~ > gets the permstatus of a user (in a channel)
    13. gettoken ~channel\currentchannel~ > gets the token of a channel (log in console if custom)

    Owner

    1. addperm ~(admin)~ ~user~ > adds admin rights to ~user~ (they cant use this command)
    2. removeperm ~(admin)~ ~user~ > removes admin rights from ~user~

    Whisper

    1. test > test cmd
    2. help ~(oauth)\standart~ > help cmd
    3. oauth ~oauth:(yourkey)/channel~ > sets the oauth key for the account or a channel if access
    4. oauth reset |~channel~ > resets the token for the account or channel if access
    5. log ~channel~ > starts logging the modlogs of a channel
    6. unlog ~channel~ > stops logging the modlogs of a channel
    7. tokeninfo ~token~ > gets info about the token of a channel
    8. checktoken ~channel~ > gets info about a token over the api
