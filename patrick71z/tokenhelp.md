# Token Help

## How to create a token the bot can use to login?

### 1. Go to https://twitchtokengenerator.com/ 

### 2. Choose "Custom Scope token" 
![1](https://user-images.githubusercontent.com/66999984/153358349-a71e30bc-7245-452b-b070-389e666a788d.png)

### 3. Scoll down until you see "channel:moderate" and "moderation:read" and click both *(You can choose the other scopes too if you want to use the token yourself)*
![2](https://user-images.githubusercontent.com/66999984/153358442-9480897f-50c0-4ace-a853-95d025b97103.png)

### 4. Scroll all the way down and click "Generate Token"
![3](https://user-images.githubusercontent.com/66999984/153358451-db2c1d38-0684-4135-875c-cfd3b0c081fe.png)

### 5. You will be redirected to a page where you have to authorize the connection, Click "Authorize" there
![4](https://user-images.githubusercontent.com/66999984/153358455-11830306-dc84-498f-afca-c2878c17f47e.png)

### 6. Done! Now you can copy the "Access Token" and use it in the bot
![5](https://user-images.githubusercontent.com/66999984/153358458-c5239b90-e2be-4d61-8f87-21dae94ffb30.png)

### Add Token to bot
#### Open a whisper message to patrick71z and send `!oauth <your token>` 
#### > Example: `!oauth bbxxpxcy1w248ajgb4rkeytdkdwffd`
