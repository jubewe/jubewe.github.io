Token Help

  1. Go to this token-generator

  2. Click on "Request Token"

  3. Click "Authorize" in the new window

  4. You will be redirected to a page which will validate the token

  5. If successful, you can click "show infos" and then on the token to copy it

    6. Add Token to bot:

    Open a whisper message to patrick71z and send !oauth ~your token~
    > Example: !oauth bbxxpxcy1w248ajgb4rkeytdkdwffd
